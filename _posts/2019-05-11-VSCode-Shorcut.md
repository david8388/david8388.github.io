將 Visual Studio Code 當作主要的開發環境，因此也必須善用這個 IDE 所給的 shortcut，讓開發速度上可以更快速。

- ⌘K ⌘S：開啟鍵盤快速鍵
- ⌘⌥ + 左右鍵：切換為左邊的 tab 視窗以及切換為右邊的 tab 視窗
- ⌘⌥ + 上下鍵：垂直選取
- ⌘P：開啟檔案
- ⌥ + 上下鍵：移動整列文字
- ⌃`：開啟終端機
- ⌃G：搜尋要瀏覽的行號
- ⌃⌘F：展開/縮小螢幕
- F12：移至定義
- ⌘K ⌘C：區塊註解
- ⌘K ⌘U：區塊註解取消
- ⌃D：選取一段詞
- ⇧⌘L ：選取檔案內所有相同文字
- ⌘/：單行註解
